# Portal de Noticias con Firebase

Un sistema completo de noticias con panel de administración protegido, utilizando Firebase para almacenamiento y autenticación.

## Características

- **Vista pública de noticias**: Los visitantes pueden ver todas las noticias
- **Vista detallada**: Al hacer clic en una noticia se muestra el contenido completo
- **Panel de administración**: Solo usuarios autenticados pueden publicar noticias
- **Autenticación segura**: Sistema de login con Firebase Authentication
- **Base de datos en tiempo real**: Firestore para almacenar las noticias
- **Diseño responsivo**: Funciona en móviles, tablets y desktop
- **Compartir noticias**: Funcionalidad para compartir noticias en redes sociales
- **Editor avanzado**: CKEditor 5 para crear contenido enriquecido
